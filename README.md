![WPF](https://img.shields.io/badge/UI-WPF-512BD4?logo=.net&logoColor=white)

![Status](https://img.shields.io/badge/Status-Active_Development-brightgreen)

**WREX** — системный менеджер для Windows, объединяющий файловый проводник, редактор реестра, мониторинг, очистку и восстановление системы в одном окне.

---

## ✨ Возможности

### 📂 Файловый менеджер
- Навигация по дискам и папкам
- Двухпанельный режим (как Total Commander)
- Live Search — мгновенная фильтрация файлов
- Панель предпросмотра — содержимое текстовых файлов
- Контекстное меню: создание, копирование, вырезание, вставка, переименование, удаление
- Работа с архивами (.zip): просмотр, извлечение, создание
- Вычисление хэш-сумм (MD5, SHA-1, SHA-256)

### 🛡️ Системный мониторинг
- Автообновление проводника (FileSystemWatcher)
- Мониторинг создания файлов и ключей реестра в реальном времени
- Трекинг установки ПО — запись всех изменений при установке программ

### 📊 Диспетчер задач
- Список процессов с обновлением каждые 3 секунды
- Статус процессов: 🔴 системные / ⚪ обычные
- Завершение, перезапуск, открытие расположения файла
- **Менеджер автозагрузки** — все записи из реестра и папок

### 📜 Редактор реестра
- Дерево ключей с ленивой загрузкой
- Создание, удаление, переименование ключей и значений
- Редактирование значений всех типов
- Поиск по реестру
- Избранные ключи
- Экспорт и сравнение веток

### 🧹 Очистка системы
- Очистка Temp, Prefetch, Recent, Windows Temp
- Статистика: удалённые файлы, папки, освобождённое место

### 🔧 Утилиты
- Быстрый запуск системных инструментов (CMD, PowerShell, regedit, services.msc и др.)
- **Менеджер служб** — запуск, остановка, перезапуск, изменение типа запуска
- **Точки восстановления** — просмотр, создание, откат
- **Восстановление Windows**:
  - Включение/отключение UAC
  - Включение Диспетчера задач, редактора реестра, CMD
  - Показ скрытых файлов и расширений
  - Сброс Windows Defender и брандмауэра
  - Восстановление системных файлов (sfc + DISM)
  - Сброс файла hosts
- **Экспорт системной информации** — отчёт о системе в файл

### 📝 История
- Журнал всех действий с фильтрацией по категориям
- Экспорт в текстовый файл
- Копирование записей

### 🖥️ Интерфейс
- Тёмная тема в стиле VS Code
- Системный трей с контекстным меню
- Автозагрузка при старте Windows
- Режим "Поверх всех окон"
- Строка состояния с информацией о системе

## 🚀 Начало работы

### Требования
- Windows 10 / 11
- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)

### Запуск из исходников
```bash
git clone https://github.com/FrazzieYT/WREX.git
cd WREX
dotnet run
```

### Сборка портативной версии
```bash
publish.bat
```
Результат в папке `publish/` — один файл WREX.exe (~60-80 MB, self-contained).

### WinRE / WinPE
Программа работает в среде восстановления Windows:
1. Скопируйте `WREX.exe` на флешку
2. Загрузитесь в WinRE
3. Откройте командную строку
4. Запустите: `X:\path\to\WREX.exe`
